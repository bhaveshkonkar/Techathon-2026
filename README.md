# Nexus 3D LMS

Nexus 3D LMS is a web-based learning management system designed to deliver immersive educational experiences using interactive 3D content, hand tracking, and AI-powered tools.

## Features
- Interactive 3D learning modules
- Hand tracking for gesture-based navigation
- AI-powered educational content generation
- Modern UI built with React and TypeScript
- Easy content management and uploads

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/nexus-3d-lms.git
   ```
2. Navigate to the project directory:
   ```bash
   cd nexus-3d-lms
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the Application
Start the development server:
```bash
npm run dev
# or
yarn dev
```

Open your browser and go to `http://localhost:3000` to view the app.

## Project Structure
- `src/` - Main source code
  - `components/` - React components
  - `lib/` - Utility functions
  - `services/` - API and AI integrations
- `uploads/` - Uploaded content
- `server.ts` - Backend server

## Usage
- Create and manage 3D educational modules
- Upload content and assets
- Use hand tracking for interactive navigation
- Generate educational content with AI
